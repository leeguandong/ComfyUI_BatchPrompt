## Install

- 推荐使用管理器 ComfyUI Manager 安装（On the Way）

- 手动安装：
    1. `cd custom_nodes`
    2. `git clone https://github.com/leeguandong/ComfyUI_BatchPrompt.git`
    3. Install dependency packages pip install -r ComfyUI_BatchPrompt/requirements.txt
    4. 重启 ComfyUI
